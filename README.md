# E-Commerce Website

A basic e-commerce website built using HTML, CSS, JavaScript, Node.js, Express.js, and MongoDB.

## Features

- User Registration
- User Login with JWT
- Product Listing
- Product Details
- Add to Cart
- Remove from Cart
- Checkout
- Order Processing
- Admin Order Dashboard
- Order Status Update

## Tech Stack

Frontend:
- HTML
- CSS
- JavaScript

Backend:
- Node.js
- Express.js

Database:
- MongoDB

## Run Backend

```bash
cd backend
node server.js
